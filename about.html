About Eris Theme

Eris is a modern, minimalist Jekyll theme with a distinctive purple color scheme and unique border styling. It features a responsive design, dropdown navigation, and social media integration.

The Story

Eris was created as a custom theme for a personal blog, with a focus on readability, minimalism, and a unique visual style. The theme is named after Eris, the Greek goddess of discord, reflecting its distinctive and somewhat unconventional design elements.

Design Philosophy

The design philosophy behind Eris is centered around these principles:

  • Minimalism: Clean, uncluttered design that puts the focus on content.
  • Distinctiveness: Unique visual elements that make your site stand out.
  • Readability: Typography and spacing optimized for reading comfort.
  • Responsiveness: Looks great on all devices, from mobile to desktop.

Features

  • Responsive design
  • Dropdown navigation
  • Social media integration
  • Custom border styling
  • Monospace typography
  • Blog post layout
  • Mobile-friendly navigation
  • Mermaid diagram support

The Creator

Eris was created by Kali Jackson, a web developer and designer with a passion for clean, functional design.
